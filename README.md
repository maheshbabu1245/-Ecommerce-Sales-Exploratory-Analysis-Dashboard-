# 📊 Power BI Sales Dashboard

## 📌 Project Overview
This project is an interactive Power BI dashboard designed to analyze business performance and generate meaningful insights from sales data.  
The dashboard helps track KPIs, monitor trends, compare regional performance, and support data-driven decision-making.

---

## 🎯 Objectives
- Analyze overall sales performance
- Track profit and revenue trends
- Identify top-performing categories and regions
- Create interactive visualizations for better business understanding
- Build a professional dashboard for reporting and analytics

---

## 🛠️ Tools & Technologies Used
- Power BI
- Excel / CSV
- DAX (Data Analysis Expressions)
- Data Cleaning
- Data Visualization

---

## 📈 Dashboard Features
- KPI Cards
- Sales Analysis
- Profit Analysis
- Interactive Filters & Slicers
- Trend Analysis
- Region-wise Insights
- Category Performance Analysis
- Dynamic Visualizations
